# coordinax.curveframes

Curve-attached reference frames for [coordinax](https://github.com/GalacticDynamics/coordinax).

This package implements moving-frame constructions for smooth curves in 3D Euclidean space, including Frenet-Serret and Bishop-style workflows.

## Highlights

- construct frames from parameterized curves
- work with tangent, normal, and binormal directions
- integrate with the broader coordinax frames and transforms APIs
